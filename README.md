# Gestore Spese Personali 💰

Applicazione da riga di comando per gestire le proprie spese personali, sviluppata in Python.

## Funzionalità
- Aggiungere transazioni (entrate e uscite)
- Visualizzare tutte le transazioni
- Visualizzare il saldo attuale
- Visualizzare le statistiche (totale entrate e uscite)

## Tecnologie
- Python 3.12

## Come si usa
1. Clona il repository
2. Esegui `python3 gestore_spese.py`
3. Segui il menu interattivo
